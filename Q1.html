What is a Perceptron?

A Perceptron is the simplest type of artificial neural network, consisting of a single-layer feed-forward network. It was introduced by Frank Rosenblatt in 1957 and serves as the fundamental building block of modern deep learning.

It has an input layer and an output layer, where the input layer receives signals and the output layer processes them to produce results. The perceptron uses weights on synaptic links to connect input neurons to output neurons, and the output is computed using an activation function.

The perceptron network consists of three types of units:

  • Sensory (Input) Units — receive external input signals
  • Associate (Hidden) Units — process intermediate computations
  • Response (Output) Units — produce the final classification result
Figure 1 — Architecture of a Single-Layer Perceptron
A perceptron receives inputs x₁…xₙ, multiplies each by its weight, sums with bias b, and passes through an activation function f to produce output y.

Binary Step Activation Function

The perceptron uses a binary step activation function with a fixed threshold (θ). The function categorizes the net input into one of two classes:

f(Yin) = { +1if   Yin > θ
 0if   −θ ≤ Yin ≤ θ
−1if   Yin < −θ
Figure 2 — Binary Step Activation Function
The binary step activation function outputs +1, 0, or −1 depending on whether the net input exceeds the threshold θ.

Steps for Training a Perceptron

The perceptron follows an iterative supervised learning algorithm to adjust its weights until the output matches the desired target. Below are the detailed steps:

1

Initialization

Set initial weights (w₁, w₂, …, wₙ) and bias (b) to zero or small random values. Define the learning rate η (usually between 0 and 1).

2

Input Assignment

Assign input values (x₁, x₂, …, xₙ) to the input layer neurons for each training sample.

3

Net Input Calculation

Compute the net input (weighted sum) using the formula:

Yin = b + Σi (xi · wi)

where xi are input values, wi are corresponding weights, and b is the bias.

4

Apply Activation Function

Use the activation function (binary step function) to calculate the output:

y = f(Yin)

where f is the binary step activation function with threshold θ.

5

Compare Output

Compare the calculated output y with the desired target output t. If y ≠ t, there is an error that must be corrected.

6

Weight & Bias Adjustment

If there is an error (y ≠ t), update the weights and bias using the Perceptron Learning Rule:

wi(new) = wi(old) + η · xi · t b(new) = b(old) + η · t

Where η is the learning rate, t is the target value (+1 or −1), and xi is the input value.

7

Repeat (Convergence)

Repeat steps 3–6 for all training patterns until the stopping condition is met — i.e., no change in weights or the error is minimized across all samples.

The training loop continues until no weight changes occur across an entire epoch — the perceptron has converged.

Key Points

  • The perceptron network consists of sensory (input), associate (hidden), and response (output) units.
  • It uses a binary step activation function with a fixed threshold.
  • The perceptron learning rule adjusts weights based on errors in the output.
  • The perceptron can only learn linearly separable patterns (e.g., AND, OR but not XOR).
  • It forms the foundation for more complex multi-layer neural networks used in deep learning.
