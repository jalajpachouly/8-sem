What is a Loss Function?

A loss function (also called cost function or objective function) is a mathematical function that measures the difference between the predicted output of a neural network and the actual target value. It quantifies the error or deviation of the model's predictions from the true values.

The goal of training a neural network is to minimize this loss, thereby improving the accuracy of the model. Loss functions guide the optimization process during training — they help the model learn by providing feedback on how well it is performing and how it can adjust its parameters (weights and biases) to reduce the error.

Types of Loss Functions

Figure 2 — Regression vs. Classification Loss Functions
Loss functions are broadly divided into regression losses (for continuous targets) and classification losses (for discrete labels/probabilities).

1. Mean Squared Error (MSE)

MSE = 1N Σi=1N (yi − ŷi)2

Measures the average squared difference between actual and predicted values. Commonly used for regression tasks.

✔ Easy to interpret   ✔ Penalizes larger errors more heavily

✗ Not robust to outliers   ✗ Error unit is squared (not intuitive)

2. Mean Absolute Error (MAE)

MAE = 1N Σi=1N |yi − ŷi|

Measures the average absolute difference between actual and predicted values. Also used for regression tasks.

✔ Robust to outliers   ✔ More intuitive error measure

✗ Does not penalize large errors as strongly as MSE

3. Binary Cross-Entropy Loss

L = − 1N Σi=1N [yi log(ŷi) + (1 − yi) log(1 − ŷi)]

Used for binary classification tasks. Measures the difference between the predicted probability distribution and actual labels.

✔ Works well for probabilistic models   ✔ Encourages predicted probability close to actual label

✗ Computationally expensive for large datasets

4. Hinge Loss

L = Σi=1N max(0, 1 − yi · ŷi)

Used for training classifiers, particularly Support Vector Machines (SVMs). Penalizes predictions that are incorrect or not confident enough.

✔ Encourages correct classification with a margin

✗ Not suitable for probabilistic models

5. Huber Loss

L = { 12(yi − ŷi)2 if |yi − ŷi| ≤ δ
δ · |yi − ŷi| − 12δ2 if |yi − ŷi| > δ

Combines MSE and MAE. Behaves like MSE for small errors and MAE for large errors. Used for regression tasks.

✔ Robust to outliers   ✔ Combines benefits of MSE and MAE

✗ Requires tuning of hyperparameter δ

6. KL Divergence Loss

DKL(P ‖ Q) = Σi P(i) log P(i)Q(i)

Measures how one probability distribution Q differs from a reference distribution P. Used in probabilistic models and generative models.

✔ Useful for comparing probability distributions

✗ Sensitive to small probabilities

7. Log-Cosh Loss

L = Σi=1N log(cosh(ŷi − yi))

Similar to MSE but less sensitive to outliers. Used for regression tasks. Smooth and differentiable everywhere.

✔ Smooth and differentiable   ✔ Robust to outliers

8. Categorical Cross-Entropy Loss

L = − Σi=1N Σj=1K yij log(ŷij)

Used for multi-class classification problems. Measures the difference between the predicted probability distribution and the actual one-hot encoded labels.

Importance of Loss Functions

  • Guides Learning — helps the model understand how far its predictions are from the actual values.
  • Optimization — used by optimization algorithms (e.g., gradient descent) to adjust model parameters.
  • Performance Metric — indicates how well the model is performing during training.
  • Problem-Specific — the choice of loss function depends on the type of problem (regression, classification, etc.) and the desired behaviour of the model.
