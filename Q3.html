What is an Activation Function?

An activation function is a mathematical function applied to the output of a neuron in a neural network. It determines whether a neuron should be activated or not, introducing non-linearity into the model.

This non-linearity allows the network to learn complex patterns and relationships in data. Without activation functions, even a deep network would behave like a single linear transformation — unable to solve problems like image recognition or NLP.

Types of Activation Functions

1. Linear Activation Function

f(x) = x

Output is directly proportional to input. Rarely used in practice — does not introduce non-linearity, so a deep network with only linear activations collapses to a single linear layer. Cannot solve complex problems.

Straight line through origin

2. Binary Step Function

f(x) = { 0if x < 0
1if x ≥ 0

Outputs 0 or 1 based on threshold at x = 0. Used in single-layer networks for binary classification. Not differentiable at x = 0, limiting its use in backpropagation.

Sharp step at x = 0

3. Bipolar Step Function

f(x) = { −1if x < 0
+1if x ≥ 0

Outputs −1 or 1. Similar to binary step but symmetric about the x-axis. Used for bipolar classification in single-layer networks.

Symmetric step: −1 or +1

4. Binary Sigmoidal Function

f(x) = 11 + e−Ax

Smooth S-curve producing output between 0 and 1. The steepness parameter A controls how sharp the curve is. Commonly used in back-propagation networks and as a probability estimator.

S-curve output: 0 to 1

5. Bipolar Sigmoidal Function

f(x) = 21 + e−Ax − 1

Smooth S-curve producing output between −1 and 1. It is a shifted and scaled version of the binary sigmoid. Also used in back-propagation networks.

S-curve output: −1 to 1

6. Tanh Activation Function

f(x) = ex − e−xex + e−x

Output range −1 to 1, zero-centred. Preferred over sigmoid for multi-layer networks as zero-centring makes optimization easier. Stronger gradients than sigmoid near x = 0.

Zero-centred, range [−1,1]

7. Hard Tanh Activation Function

f(x) = { −1if x < −1
xif −1 ≤ x ≤ 1
+1if x > 1

A computationally cheaper piecewise linear approximation of tanh. Uses clipping — outputs −1 or +1 at extremes and identity in between.

Piecewise linear clipping

8. ReLU (Rectified Linear Unit)

f(x) = max(0, x)

Outputs 0 for all negative inputs and the raw value for positives. Most widely used activation in deep networks — computationally cheap, avoids the vanishing gradient problem, and enables fast convergence.

Zero for x<0, linear for x≥0

9. Softmax Activation Function

f(xi) = exiΣj exj

Converts a vector of real numbers into a probability distribution (all outputs sum to 1). Used exclusively in the output layer for multi-class classification. Larger inputs get proportionally higher probabilities.

Output sums to 1 across all classes

Importance of Activation Functions

  • Introduce non-linearity, enabling the network to learn complex, real-world patterns.
  • Determine whether a neuron should fire or remain dormant, controlling information flow.
  • Play a crucial role in the convergence speed of the model during training.
  • Choice of activation function directly impacts susceptibility to the vanishing/exploding gradient problem.
